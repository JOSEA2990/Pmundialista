Al registrarse cada participante recibirá un ID único en formato UXXXXXX, el cual permitirá acceder a sus pronósticos.

2️⃣ Sistema de Puntuación

Fase de Grupos
  • 3 puntos → acertar ganador o empate
  • 5 puntos → resultado exacto
Fase Eliminatoria
  • 3 puntos → acertar equipo que avanza
  • +3 puntos → resultado exacto
  • +2 puntos → acertar que el partido se va a penales
